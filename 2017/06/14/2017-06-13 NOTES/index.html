#usermod#passwd#mkpasswd
Jun 14, 2017
usermod命令

语法

usermod -u 改uid

usermod -g 改用户组

usermod -G 扩展组

usermod -d 改用户家目录

usermod -s 改用户shell

用户密码管理

语法

passwd root用户更改密码

passwd 用户名 普通用户更改密码

passwd -l 用户名 锁定用户

passwd -u 用户名 解锁用户

asermod -L 用户名 锁定用户

asermod -U 用户名 解锁用户

passwd --stdin 用户名 给特定用户更改密码

echo “密码” |passwd --stdin 用户名 脚本更改特定用户密码

echo -e “密码”\n密码” |passwd 用户名 可以在一行命令里面出现换行符



设定密码长度 最好超10位 包括大小写 数字 特殊字符 没有规律性

mkpasswd工具

mkpasswd可以生成密码工具

首先需要安装工具

yum install -y expect

语法

mkpasswd 默认生成一个9位密码

mkpasswd -l 12 生成一个12位密码

mkpasswd -l 12 -s 3 生成一个12位密码 包括3个特殊符号
