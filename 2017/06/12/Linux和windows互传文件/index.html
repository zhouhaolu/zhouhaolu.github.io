文件互传

#linux#Windows
Jun 12, 2017

lrzsz是一个unix通信套件提供的X,Y,和ZModem文件传输协议可代替ftp上传和下载.需要注意的事这个命令无法在putty界面使用.

首先安装lrzsz

yum -y install lrzsz

上传文件，执行命令rz，会跳出文件选择窗口，选择好文件，点击确认即可。
rz

下载文件，执行命令sz
sz
