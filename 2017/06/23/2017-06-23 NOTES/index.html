#zip#tar#打包并压缩
Jun 23, 2017
zip压缩工具

yum install -y zip（安装zip压缩）

yum install -y unzip（安装解压缩）

zip 1.txt.zip 1.tx （压缩文件）

zip -r 1.zip 2.txt file （压缩文件+目录）（zip压缩文件文件后源文件不删除）

unzip 压缩文件名称 （解压缩到当前目录下）

unzip 1.txt.zip -d test/（解压1.txt到test目录）



不能查看文件内容

文件压缩后不删除源文件

tar打包工具

tar -cvf txt.tar test/ （打包test下的所有文件名称为txt.tar）

tar xvf txt.tar(解包)

tar -tftxt.tar（查看打包内容）

tar -cvf test.tar --exclude 1.xt 2.txt test1 (打包并过滤1.txt、2.txt、test1文件)

tar打包并压缩

tar -czvf test.tar.gz test1 （gzip 打包压缩）

tar -cjvf test.tar.bz2 test1 （bz2 打包压缩）

tar -cJvf test.tar.xz test1 （xz 打包压缩）

解压缩时将c换成x
