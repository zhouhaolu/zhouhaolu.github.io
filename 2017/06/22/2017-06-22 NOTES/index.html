#gzip#bzip#xz
Jun 22, 2017
常见压缩文件
  • Windows .rar .zip .7z
  • Linux .zip .gz .bz2 .tar.gz .tar.bz2 .tar.xz
gzip压缩工具

gzip 文件名称 （压缩文件）

gzip -1 -9 （压缩等级,默认为6级别）

gzip -d 文件名称 （解压缩）

gzip -c 文件名称

  • gzip -c 1.txt > /tmp/1.txt.gz (压缩的时候指定到一个文件路径并且源文件不消失)

  • gzip -d -c 1.txt /tmp///1.txt.gz > /tmp /1/2.txt (解压缩到指定文件夹且源文件不消失)

file 压缩文件名称 （查看压缩文件信息）

zcat 压缩文件名称 （查看压缩文件内容）



gzip 不能压缩目录

bzip2压缩工具

yum install -y bzip2 （安装bzip2）

bzip2 文件名称 （压缩文件）（相比gzip 压缩更狠）

bzip2 -d 文件名称 （解压文件）

bzcat 压缩文件名称 （查看压缩文件内容）



bzip2 不能压缩目录

xz压缩工具

xz 文件名称 （压缩文件）（比bzip2压缩更狠）

xz -d 文件名称 （解压缩）

unxz 文件名称 （解压缩）
