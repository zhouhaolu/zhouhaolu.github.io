#rpm#yum#yum本地仓库
Jun 26, 2017

rpm

rpm -* 包名

  • rpm -ivh 1.rpm （安装rpm包）
  • rpm -Uvh 1.rpm (升级rpm包文件)
  • rpm -e 1（卸载rpm包）
  • rpm -qa (查询安装的包)
  • rpm -q（查询某个包是某安装）
  • rpm -qi (查询指定包信息)
  • rpm -ql (列出包安装文件)
  • rpm -qf 文件绝对路径 （查看一个文件由是哪个包安装的）

yum

yum *包名称

  • yum list （列出可用的rpm包）
  • yum search （搜索包）
  • yum install -y (安装包)
  • yum grouplist（列出安装组）
  • yum groupinstall ‘xx’(安装组)
  • yum remove (卸载包)
    yum upate -y`(跟包名升级包 不跟东西 升级所有东西)
  • yum provides "/*/***“
